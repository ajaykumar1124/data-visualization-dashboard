## 🔧 **REST API FEATURES**

### **🔐 Authentication & Security**
- JWT token-based authentication
- Role-based access control (User, Analyst, Admin)
- API key support for automation
- Rate limiting and security middleware

### **📊 Data Management Endpoints**
```
POST /api/v1/data/upload          # Upload CSV/Excel files
POST /api/v1/data/process/{id}    # Run cleaning pipeline
GET  /api/v1/data/datasets        # List user datasets
GET  /api/v1/data/datasets/{id}   # Get dataset info
DELETE /api/v1/data/datasets/{id} # Delete dataset
```

### **📈 Visualization Endpoints**
```
POST /api/v1/visualizations/create/{dataset_id}  # Create charts
GET  /api/v1/visualizations/dataset/{dataset_id} # List charts
GET  /api/v1/visualizations/{viz_id}             # Get chart data
```

### **📤 Export & Reporting**
```
POST /api/v1/export/create/{dataset_id}  # Create export job
GET  /api/v1/export/download/{filename}  # Download files
```
